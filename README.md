# MULTI USER CHAT WEB

Chat Web allows multi user to communicate each other.

```
dependencies": {
    "express"
    "socket.io"
}
```

## Installation

```
npm i
```

## Usage

To connect server

```
node server.js
```

http://localhost:3000/
